## Project Overview

Number Sense Suite is an in-browser toolkit for assessing and training dyscalculia-related skills. Everything runs locally: the UI loads from `index.html`, `scripts/main.js` drives tabbed flows for Assessment, Adaptive Coaching, inline Training drills, Cognition batteries, and a Progress dashboard. Results (assessments, drills, cognition runs, adaptive plan weights) live in `localStorage` under the `ns-suite-v2` key and never leave the browser. Chart.js renders longitudinal accuracy trends, while CSV/JSON export buttons give caregivers or researchers snapshots of the captured data.
