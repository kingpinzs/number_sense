# Discalculas

[![CI](https://github.com/JeremyDuncan/discalculas/actions/workflows/ci.yml/badge.svg)](https://github.com/JeremyDuncan/discalculas/actions/workflows/ci.yml)

Offline-first dyscalculia training companion built with Vite + React 19 + TypeScript.

## Features

- Mobile-first responsive design
- Offline-first PWA architecture
- Adaptive difficulty training
- Progress tracking and analytics
- WCAG 2.1 AA accessible

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run tests
npm run test

# Run E2E tests
npm run test:e2e

# Build for production
npm run build
```

## Tech Stack

- **Framework:** React 19.2 + TypeScript 5.9
- **Build Tool:** Vite 7.2
- **Styling:** Tailwind CSS v4 + shadcn/ui
- **Database:** Dexie (IndexedDB)
- **Testing:** Vitest + React Testing Library + Playwright
- **Animation:** Framer Motion
